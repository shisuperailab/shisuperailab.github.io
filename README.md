# Shi Group Website

This repository contains the website for the Shi Group, the AI-driven (Bio)Macromolecular Modeling and Design Group at the University of Toledo.

Website: https://shisuperailab.github.io/

## Local Preview

Run a local static server from the repository root:

```bash
python3 -m http.server 8001
```

Then open:

```text
http://127.0.0.1:8001/
```

## Structure

- `index.html` - home page
- `people.html` - team page
- `research.html` - research thrusts
- `publications.html` - publications
- `openings.html` - open positions
- `contact.html` - contact information
- `assets/css/group.css` - main group-site styling
- `robots.txt`, `sitemap.xml`, `llms.txt` - indexing and AI-search metadata

## Deployment

The site is deployed with GitHub Pages from:

```text
shisuperailab/shisuperailab.github.io
```
